Whisper Web Transcriber
======================

โปรเจกต์นี้เป็นเว็บแอปสำหรับแปลงไฟล์เสียงเป็นข้อความ
โดยใช้ Flask เป็น API และ OpenAI Whisper เป็นตัวถอดเสียง
รองรับไฟล์ mp3, wav, m4a และ mp4


โครงสร้างการทำงาน
-----------------
Client (HTML + JS)
→ ส่งไฟล์เสียง
→ Flask API
→ แปลงไฟล์เป็น wav (ถ้าจำเป็น)
→ Whisper ถอดเสียง
→ ส่งข้อความกลับไปแสดงบนเว็บ


เครื่องมือที่ใช้
---------------
- Python 3.9+
- Flask
- OpenAI Whisper
- ffmpeg
- HTML / JavaScript


การติดตั้ง
----------
1. ติดตั้ง Python
   https://www.python.org/

2. ติดตั้ง ffmpeg และเพิ่มเข้า PATH
   ตรวจสอบด้วยคำสั่ง
   ffmpeg -version

3. ติดตั้งไลบรารีที่จำเป็น
   pip install flask openai-whisper torch


วิธีรันเซิร์ฟเวอร์
-----------------
1. เปิดโฟลเดอร์โปรเจกต์
2. รันคำสั่ง
   python server.py

3. เซิร์ฟเวอร์จะรันที่
   http://localhost:5000


วิธีใช้งาน
---------
1. เปิดไฟล์ index.html ผ่าน Browser
2. เลือกไฟล์เสียงจากเครื่อง
3. กดปุ่ม "เริ่มถอดเสียง"
4. ระบบจะแสดงสถานะ
   - กำลังส่งไฟล์
   - กำลังถอดเสียง
   - เสร็จสิ้น
5. ข้อความที่ถอดเสียงได้จะแสดงบนหน้าเว็บ


API ที่ใช้
---------
POST /transcribe
- ใช้สำหรับอัปโหลดไฟล์เสียง
- รับไฟล์ผ่าน form-data (key: file)

GET /status
- ใช้ตรวจสอบสถานะการทำงาน
- ส่งกลับ status และ text


สถานะของระบบ
-------------
idle       : ยังไม่มีงาน
listening  : กำลังถอดเสียง
done       : ถอดเสียงเสร็จแล้ว
error      : เกิดข้อผิดพลาด


หมายเหตุ
--------
- ไฟล์เสียงขนาดใหญ่จะใช้เวลานาน
- ใช้ CPU อาจช้ากว่า GPU
- ระบบรองรับการประมวลผลครั้งละ 1 งาน
